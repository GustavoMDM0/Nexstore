### 📖 Sobre o Projeto
O **Nexstore** é uma plataforma robusta desenvolvida para a **EletroCelo** gerenciar seu fluxo de produtos e vendas. O sistema integra o controle de estoque com o ponto de venda (PDV), garantindo que a disponibilidade dos produtos seja atualizada automaticamente a cada transação.

### 🌟 Funcionalidades Principais
- **✅ Controle de Estoque (CRUD):** Gestão completa de produtos com monitoramento de nível mínimo.
- **💰 Lógica de Vendas Automatizada:** Sistema de checkout que valida disponibilidade e abate estoque em tempo real.
- **📊 Dashboard Analytics:** API preparada para fornecer métricas de valor total em estoque e alertas de reposição.
- **🔐 Segurança e Performance:** Uso de variáveis de ambiente (`.env`), padrões RESTful e conexões seguras com o Atlas.
